📝 INSTRUCTIONS


Welcome to Guess What. This is a single player guessing game.



CONTROLS:


🎲 DIALOG BOX -> This is where you insert the number that you have guessed.
📥 CHECK -> Press to check if the entered number is same as the secret number.
🔄 PLAY AGAIN! -> Reset the scores and start a new game.


HOW TO PLAY:


1. You get 7 guesses to find the secret number hidden behind the question mark at the middle.
2. With each guess, you enter an integer number between 1 and 20 (both inclusive).
3. Click on the `CHECK` button or press enter to check if your guess is correct.
4. You start with 70 points and they decreses with every consequtive guess to consume.
5. You lose once your score reaches zero.
